# Tabela Hash

## Conceito
- Estrutura de dados que mapeia chave → valor.
- Usa função de hash para encontrar a posição de armazenamento.

## Problemas
- Colisões: duas chaves diferentes com mesmo hash.
- Soluções:
  - Encadeamento.
  - Endereçamento aberto.

## Complexidade
- Busca: O(1) média, O(n) pior caso.
