# Fila (Queue)

## Conceito
- Estrutura FIFO (First In, First Out).
- Operações:
  - enqueue: insere no fim.
  - dequeue: remove do início.

## Vantagens
- Útil para processamento em ordem.
- Fácil implementação com listas ou arrays circulares.

## Aplicações
- Impressoras.
- Processos em sistemas operacionais.
