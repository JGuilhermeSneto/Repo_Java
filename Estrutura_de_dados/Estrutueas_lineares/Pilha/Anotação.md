# Pilha (Stack)

## Conceito
- Estrutura LIFO (Last In, First Out).
- Operações:
  - push: inserir no topo.
  - pop: remover do topo.
  - peek/top: ver o elemento no topo.

## Vantagens
- Simples de implementar.
- Útil para desfazer ações (Ctrl+Z), navegação de páginas.

## Complexidade
- Inserção: O(1)
- Remoção: O(1)
- Acesso: O(n) no pior caso.
