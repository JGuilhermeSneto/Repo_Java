# Array (Vetor e Matriz)

## Definição
- Estrutura de dados que armazena elementos do mesmo tipo em posições contíguas na memória.
- Acesso por índice (O(1)).

## Tipos
- Vetor: 1 dimensão.
- Matriz: 2 ou mais dimensões.

## Vantagens
- Acesso rápido.
- Estrutura simples.

## Desvantagens
- Tamanho fixo.
- Inserções/remoções custosas.

## Exemplo (Python)
```python
vetor = [10, 20, 30]
print(vetor[1])  # 20
